# canvodpy

Umbrella package providing unified access to all canVOD packages.

## Installation

```bash
uv pip install canvodpy
```

## Usage

```python
import canvodpy

# Access all subpackages
from canvod.readers import Rnxv3Obs
from canvod.grids import HemiGrid
```
